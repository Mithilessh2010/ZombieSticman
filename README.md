# Stickman: Last Stand

A 2D side-view zombie survival game built for the web. Survive endless waves of zombies, earn coins, unlock weapons, and upgrade your loadout to last as long as possible.

## Gameplay

You play as a stickman fighting against waves of zombies coming from both sides. Each wave gets harder, and you earn XP and coins by defeating enemies.

Between waves, you can:
- Buy new weapons in the shop
- Equip weapons from your inventory
- Upgrade your character

The goal is simple: survive as many waves as possible.

## Controls

- A / D or Arrow Keys → Move left and right  
- W / Up Arrow / Space → Jump  
- Mouse Click → Shoot (or auto-shoot depending on settings)

## Features

- Wave-based zombie survival system
- Multiple zombie types with increasing difficulty
- Weapon shop with unlockable guns
- Inventory system to equip weapons
- XP and upgrade system
- Coins for progression
- Simple stickman-style visuals
- Smooth 2D canvas gameplay

## Weapons

- Pistol (starter weapon)
- SMG
- Shotgun
- Rifle
- Sniper

Each weapon has different stats like damage, fire rate, and reload speed.

## Progression

- Kill zombies to earn XP and coins
- Level up to get upgrades
- Use coins to buy weapons in the shop
- Equip weapons from inventory before starting runs
